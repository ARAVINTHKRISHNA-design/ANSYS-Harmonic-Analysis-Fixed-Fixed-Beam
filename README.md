ANSYS Harmonic Analysis of Fixed-Fixed Beam
Project Overview

This project performs harmonic response analysis on a fixed-fixed steel beam subjected to dynamic loading from two machines mounted at one-third locations from each end.

The objective is to determine the vibration response, resonance frequencies, deformation amplitudes, and phase angle variations within the operating speed range of the machines.

Problem Statement
Beam Length: 3 m
Beam Width: 0.5 m
Beam Thickness: 0.025 m
Material: Structural Steel
Machine Locations: 1/3 span from both ends
Harmonic Load: 250 N at each machine location
Operating Speed Range: 30–3000 RPM
Damping Ratio: 2%
Software Used
ANSYS Workbench 2025 R2 Student
Harmonic Response Analysis
Modal Analysis (Pre-Stress Modal)
Methodology
Create beam geometry.
Assign structural steel material.
Apply fixed support at both ends.
Perform modal analysis.
Transfer modal results to harmonic response system.
Apply harmonic loads at machine mounting locations.
Define frequency sweep corresponding to machine RPM range.
Evaluate frequency response and phase angle.
Results
Peak response observed near resonance frequency.
Maximum deformation amplitude ≈ 4.96 mm.
Resonance occurred around 14.7 Hz.
Significant phase shift observed near resonance.
Response amplitude decreases after resonance due to damping effects.
